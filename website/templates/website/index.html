What is Kaboom?

Kaboom is a database of comics and cartoons. It contains metadata about comic issues, cartoon episodes, characters and so on (you can see the full list below). You can also create an account and track the issues and episodes that you have watched/read.

So, how do I use Kaboom?

As mentioned above, Kaboom is just a database. You need to use a client to actually access the data (for now), there are a list of clients below. If you want to make a client, check out the docs.

This is the Kaboom staging instance and is only used for client and API testing. Head over to the live instance.
