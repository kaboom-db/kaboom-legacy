What is Kaboom?

{% endif %}

Kaboom is a database of comics and cartoons. It contains metadata about comic issues, cartoon episodes, characters and so on (you can see the full list below). You can also create an account and track the issues and episodes that you have watched/read.

So, how do I use Kaboom?

As mentioned above, Kaboom is just a database. You need to use an app to actually access the data (for now), there are a list of apps below. If you want to make an app, check out the docs.
