Here are some things to keep in mind when approving an image:

  • No sexually explicit content is allowed.
  • Each record must be valid - for example you cannot submit a logo type for an episode.
  • Make sure that the object id actually exists in the database (it will throw an error if it doesn't anyway).
  • Try to accept images with acceptable aspect ratios - for example a logo should be 1:1 aspect ratio with no background.
Have a look at the data contribution guidelines if you are unsure, or feel free to contact one of the other admins.
